# AgentX - Next Gen AI Interface

A premium, glassmorphic React dashboard for the AgentX framework.

## Features
- **Modern UI:** Glassmorphism-based design with blur effects and sleek animations.
- **Real-time Reasoning Hub:** Visual tracking of agent's thinking process.
- **Session Management:** Browse and switch between multiple agent conversations.
- **Markdown Rendering:** Full support for rich text and code blocks.

## Development

### Prerequisites
Ensure the AgentX backend is running (typically on port 3000).

### Setup
1. `npm install`
2. `npm run dev`

### API Proxy
The project is configured to proxy `/api` requests to `http://localhost:3000`. You can adjust this in `vite.config.ts`.

## Tech Stack
- **React 18+**
- **Vite**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **Vanilla CSS** (Design System)
